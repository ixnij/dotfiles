* Please place the non-melpa packages in /site-lisp/.
  Managing the non-melpa packages by using git submodule.

  For example:

  #+BEGIN_SRC bash
  git submodule add https://github.com/foo/bar.git site-lisp/bar
  git submodule init
  git submodule update
  #+END_SRC
