## Emacs

### Requirements

* `stow`
* `SF Mono`
* `emacs`>= 29 with patches from emacs-plus

### Quickstart

```console
$ stow ${repository}/emacs -t ~/.emacs.d/ # make sure your ~/.emacs.d/ exists
$ emacs --daemon
```
